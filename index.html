What is an 'event listener'?
A method that attaches an event handler to a specified element. The result is that the function is performed only if the user performs a specified action such as clicking a button or scrolling. It will contain an event, a function and a userCapture.
How do you attach an event listener that responds to a 'click' event to an element using jQuery?
By adding the event called .click() you bind or trigger the handler to the specified event. You can also use a longer version which is .on( "click", handler). Any HTML element can receive this event.
What is the ID of the place where the output of the loop function should go?
After writing the loop, any statements contained within the curly braces will be executed as specified by the loop. In this program, the ID of the place where the output of the loop function will go is #output_area. It is further specified as involving jQuery with the $ sign.
